# LinuxCurse
Install any modpack from CurseForge on linux easily

<h1>How to use</h1>

<p>Open terminal and paste these commands:</p>
<p>cd Downloads</p>
<p>git clone https://github.com/CustomCheat/LinuxCurse.git</p>
<p>cd LinuxCurse</p>
<p>pip install -r requirements.txt</p>
<p>python3 main.py</p>
